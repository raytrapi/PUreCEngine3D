# Motor videojuegos
Motor videojuegos en 2D/3D. Este es un proyecto de investigación para ver nuestras capacidades a la hora de desarrollar un gran proyecto. 
Para agilizar el funcionamiento del proyecto, este se desarrollará en paralelo con otros proyectos de juego. De esta forma no nos centraremos en la creación solo del editor, sino que ayudaremos al desarrollo de la parte funcional.


## Requisitos
Actualmente se requiere de **conan**, pero no es fundamental.
- OpenGL: Se requiere al menos la versión 2.0 pero está adaptada a la versión 4.6
